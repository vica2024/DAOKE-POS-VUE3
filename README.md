## 安装使用

- 获取代码
```
git clone https://github.com/vica2024/vica-retail-cashier.git
```
- 安装依赖
```
cd vica-retail-cashier && yarn install 
```
或者
```
cd vica-retail-cashier && npm install
```
- 运行项目
```
yarn dev
```
或者
```
nup run dev
```
- 打包发布
```
yarn build
```
